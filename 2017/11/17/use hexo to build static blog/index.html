Layout

Layout原意是布局的意思,在HEXO中,包含三种文件:post,pages和draft.

  1. post布局生成的路径:source/_posts,使用该布局生成的post都是在页面中所要展现出来的
  2. draft布局生成的路径:source/_drafts,这个布局表示草稿的意思,即该目录下的所有post都当做是草稿,不会展现到页面中,如果需要展示可以修改_config.yml中的render_drafts为true,这样可以将草稿也展示到页面中.
  3. page布局 表示一个脚手架的例子,我们可以在scaffolds添加一个模板,叫page,然后通过new来创建一个模板的post
